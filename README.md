## 2. 命令
### 2.1 生成文章：
```
 hugo new posts/文件夹/index.zh-cn.md  
```
 还需要一个featured-image.png 当做封面

 ### 2.2 本地运行
 ```
hugo serve --disableFastRender //能够实时预览你正在编辑的文章页面
```
文件内容更改时, 页面会随着更改自动刷新,但是可能需要时间来编译响应.
